# Introdução ao GIT

- Inicializar o projeto (repositório local)

```bash
$ git init
```

- Descobrir o estado (status) do repositório:

```bash
$ git status
```

- Para adicionar novas mudanças:

```bash
$ git add <nomedoarquivo>
```

- Para gravar as mudanças:

```bash
$ git commit -m "Message"
```
